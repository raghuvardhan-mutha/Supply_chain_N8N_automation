# Reference Material

This folder holds supplementary files that informed this project but aren't part of the core n8n → Supabase → Quadratic pipeline.

## `amount_reimbursed_powerbi_reference.pbix`

A Power BI file used as background reference while learning dashboard-building patterns (amount-reimbursed/expense-style reporting). It uses a different dataset than the supply chain project and is kept here only as supporting material — see [`../docs/09_Future_Enhancements.md`](../docs/09_Future_Enhancements.md) for how a Power BI layer could extend this project's own dashboard in the future.

Open it with [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (Windows only).
